2025.01 時点での、Windows上での学習が主体となっており、技術的細部については不明のまま実装している点もある

- React + Redux + RTK + QTK Query
- Aspida (Axios)
- Chakura UI
- React Router
- サンプル
  - Redux + RTK の実装例（Counter）
  - RTK Query + Aspida の実装例 (Weather)
  - オフィシャルプラグインの組み込み例 (Os)
